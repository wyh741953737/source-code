# sourcing-h5

#### 项目启动
```
yarn && yarn start
```

#### 目录结构
```
sourcing-h5
  ├── mock                        // mock
  ├── src                         // 项目资源
  │   ├── .umi                    // umi
  │   ├── assets                  // 静态资源
  │   ├── components              // 组件
  │   ├── config                  // 环境配置包括app端
  │   ├── hooks                   // hook
  │   ├── pages                   // h5页面/app屏幕
  │   ├── services                // 接口API
  │   ├── styles                  // 全局样式
  │   ├── utils                   // 工具包
  │   │   ├── constant            // 枚举类
  │   │   ├── jsTracking          // 埋点
  │   │   ├── location            // 页面跳转
  │   │   ├── px2dp               // app屏幕适配
  │   │   ├── request             // 请求
  │   │   └── storage             // 本地存储
  │   │   └── tools               // 工具方法
  │   └── global.less             // 全局样式文件
  ├── i18n.util.config            // i18工具配置文件
  ├── .umirc.ts                   // umi配置文件
  ├── App.tsx                     // 入口文件
  ├── Readme.md                   // help
  └── package.josn                // package
```
